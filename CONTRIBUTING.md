# 🤝 Contributing to vmix

We love contributions! `vmix` is designed to be highly modular and extensible.

## 🛠️ How to Help
Current areas for improvement:
- **Linux Backend**: Implementing PulseAudio/PipeWire support via `pulsectl`.
- **macOS Backend**: Native CoreAudio support via `pyobjc`.
- **UI Enhancements**: New themes or layout options.

## 🚀 Getting Started
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/cool-new-thing`).
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request!

### Coding Style
- Follow PEP 8 for Python code.
- Keep the UI minimal and consistent with the existing aesthetic.
- Ensure all logic is cross-platform where possible (or abstract it into the `Backend` class).
